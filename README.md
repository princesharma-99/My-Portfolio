# 🌐 My Portfolio

A fully responsive and modern personal portfolio website built using HTML, CSS, JavaScript, and Bootstrap.
It showcases my skills, certificates, projects, and contact information with a clean dark-themed UI and smooth user experience.
Designed to work perfectly on mobile, tablet, and desktop devices.

## 🌟 Features
✨ Modern dark-themed responsive UI  
🧑 About Me section with animated typing effect  
💻 Skills showcase with modern cards  
📜 Certificates slider using Bootstrap carousel  
🚀 Projects section with live demo links  
📬 Contact form with social media integration  
🎨 Smooth animations and hover effects  
📱 Mobile-friendly responsive layout
